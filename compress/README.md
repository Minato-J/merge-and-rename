# 🗜 图片压缩工具箱

一个轻量级的 Python 批量图片有损压缩工具，支持 **JPEG 质量调节**、**分辨率缩放** 与 **WebP 格式转换**。

输出到源文件夹内的 `compressed/` 子文件夹，**绝不覆盖原始文件**。

## ✨ 功能

- **质量压缩** — 调整 JPEG/WebP 输出质量（1-100），有效减小文件体积
- **分辨率缩放** — 按百分比缩小图片尺寸（如 50% 即宽高各减半）
- **格式转换** — 支持保持原格式、统一转为 JPEG 或 WebP
- **智能处理** — PNG 透明通道自动填充白色背景，GIF 取首帧
- **自然排序** — 按数字大小而非字典序排列（如 2 排在 10 前面）
- **双模式操作** — 可拖拽文件夹到 bat 脚本，也可在终端交互菜单中操作
- **安全输出** — 压缩结果输出到子文件夹，原始图片不受影响

## 📁 项目结构

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r ../requirements.txt
```

唯一依赖：[Pillow](https://python-pillow.org/) >= 10.0.0

### 2. 运行方式

#### 方式一：双击 bat 脚本（推荐）

| 脚本 | 用途 |
|------|------|
| `图片压缩.bat` | 双击打开交互菜单；拖拽文件夹到脚本上可直接压缩 |

#### 方式二：命令行

```bash
# 打开交互菜单
python image_compressor.py

# 快速压缩（使用默认质量 85、原始尺寸、保持原格式）
python image_compressor.py "D:\图片文件夹"

# 指定压缩质量（如质量 50）
python image_compressor.py "D:\图片文件夹" 50
```

## 🎨 压缩选项说明

### 压缩质量 (1-100)

仅对 JPEG 和 WebP 输出生效。数值越低文件越小但画质越差：

| 质量 | 效果 | 适用场景 |
|------|------|----------|
| 90-100 | 几乎无损，文件较大 | 打印、存档 |
| 70-85 | 肉眼几乎无差，文件明显减小 | 日常使用（默认） |
| 40-60 | 轻微画质损失，文件大幅减小 | 网页、社交媒体 |
| 10-30 | 明显画质损失，文件极小 | 缩略图、预览 |

### 缩放比例 (1-100%)

- 100% = 保持原始分辨率
- 50% = 宽高各缩小一半（像素数变为原来的 1/4）
- 缩放采用高质量 Lanczos 算法，保证清晰度

### 输出格式

| 选项 | 说明 |
|------|------|
| 保持原格式 | PNG/GIF/BMP/TIFF 自动转为 JPEG 以应用压缩 |
| JPEG (.jpg) | 所有图片统一输出为 JPEG，透明通道自动填白 |
| WebP (.webp) | 所有图片统一输出为 WebP，兼顾小体积与透明度 |

## 📦 支持的输入格式
