# 📷 图片工具箱

一个轻量级的 Python 图片批处理工具箱，支持 **竖向拼接** 与 **智能重命名**。

## ✨ 功能

- **居中竖向拼接** — 将文件夹内所有图片按文件名自然排序，居中竖向拼接为一张透明背景的 PNG
- **智能重命名** — 自动清理文件名、格式化数字后缀（如 `img 5` → `img05`）
- **自然排序** — 按数字大小而非字典序排列（如 2 排在 10 前面）
- **双模式操作** — 可直接拖拽文件夹到 bat 脚本，也可在终端交互菜单中操作

## 📁 项目结构

```
merge/
├── image_merger.py      # 核心 Python 脚本（拼接 + 重命名）
├── 图片拼接.bat          # Windows 快捷启动脚本（支持拖拽）
├── RENAME.bat            # Windows 重命名快捷脚本
├── requirements.txt      # Python 依赖
├── test_images/          # 测试图片文件夹
└── README.md
```

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

唯一依赖：[Pillow](https://python-pillow.org/) >= 10.0.0

### 2. 运行方式

#### 方式一：双击 bat 脚本（推荐）

| 脚本 | 用途 |
|------|------|
| `图片拼接.bat` | 双击打开交互菜单；拖拽文件夹到脚本上可直接拼接 |
| `RENAME.bat` | 双击打开智能重命名菜单 |

#### 方式二：命令行

```bash
# 打开交互菜单
python image_merger.py

# 直接拼接（使用默认输出文件名 merged_output.png）
python image_merger.py "D:\图片文件夹"

# 指定输出文件名
python image_merger.py "D:\图片文件夹" "我的拼接图.png"
```

### 3. 交互菜单说明

```
==================================================
         📷 图片工具箱
==================================================
  当前文件夹: D:\图片文件夹
  输出文件名: merged_output.png
--------------------------------------------------
  1. 选择文件夹
  2. 预览图片列表
  3. 设置输出文件名
  4. 🚀 开始拼接
  5. 🔧 智能重命名
  0. 退出
--------------------------------------------------
```

## 🎨 拼接说明

- 支持的输入格式：`.png` `.jpg` `.jpeg` `.bmp` `.gif` `.webp` `.tiff` `.tif`
- 输出格式：**PNG**（RGBA 透明背景）
- 所有图片按文件名 **自然排序** 依次纵向排列
- 宽度较窄的图片会 **居中对齐**，两侧填充透明像素
- 保留图片的透明通道

## 🔧 重命名说明

智能重命名会执行以下操作：

- 去除文件名 **首尾空格**
- 去除前缀末尾的 **空格、下划线、横杠**
- 数字后缀格式化为 **两位数**（如 `photo 5.jpg` → `photo05.jpg`）
- 无数字后缀的文件自动追加 `01`（如 `cover.jpg` → `cover01.jpg`）

提供 **预览模式** 和 **正式重命名** 两种模式，操作前可先预览结果。
